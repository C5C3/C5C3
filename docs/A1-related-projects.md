# Related Projects

CobaltCore integrates and utilizes a number of external open-source projects. The following table lists runtime dependencies, build tools, and related projects. Projects in **bold** are core runtime dependencies or CobaltCore-developed components.

| Project                                                                                      | Description                                                                                                                           | Repository                                 |
| -------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
| **[Aurora Dashboard](https://github.com/cobaltcore-dev/aurora-dashboard)**                   | Unified management UI for CobaltCore                                                                                                  | `cobaltcore-dev/aurora-dashboard`          |
| **[Cortex](https://github.com/cobaltcore-dev/cortex)**                                       | Intelligent multi-domain scheduler (see [Cortex Scheduling](./08-cortex-scheduling.md))                                               | `cobaltcore-dev/cortex`                    |
| [cosign](https://github.com/sigstore/cosign)                                                 | Container image signing and verification (Sigstore project)                                                                           | `sigstore/cosign`                          |
| [Crossplane](https://github.com/crossplane/crossplane)                                       | Control plane framework for full-stack infrastructure provisioning (CNCF Incubating; see [Crossplane](./12-crossplane/))              | `crossplane/crossplane`                    |
| **[External Arbiter Operator](https://github.com/cobaltcore-dev/external-arbiter-operator)** | Ceph stretched cluster arbiter management                                                                                             | `cobaltcore-dev/external-arbiter-operator` |
| **[Flux Operator](https://github.com/controlplaneio-fluxcd/flux-operator)**                  | Kubernetes operator for declarative FluxCD lifecycle management via FluxInstance CRD                                                  | `controlplaneio-fluxcd/flux-operator`      |
| **[FluxCD](https://github.com/fluxcd/flux2)**                                                | GitOps-based continuous delivery for multi-cluster management (CNCF Graduated; see [GitOps](./11-gitops-fluxcd/))                     | `fluxcd/flux2`                             |
| **[Gardener](https://github.com/gardener/gardener)**                                         | Kubernetes cluster management                                                                                                         | `gardener/gardener`                        |
| **[GardenLinux](https://github.com/gardenlinux/gardenlinux)**                                | Optimized Linux for hypervisor nodes                                                                                                  | `gardenlinux/gardenlinux`                  |
| **[Greenhouse](https://github.com/cloudoperators/greenhouse)**                               | Centralized monitoring and alerting platform (optional)                                                                               | `cloudoperators/greenhouse`                |
| [Grype](https://github.com/anchore/grype)                                                    | Vulnerability scanner for container images and filesystems                                                                            | `anchore/grype`                            |
| **[IronCore](https://github.com/ironcore-dev/ironcore)**                                     | Bare-metal provisioning and management                                                                                                | `ironcore-dev/ironcore`                    |
| **[K-ORC](https://github.com/k-orc/openstack-resource-controller)**                          | Kubernetes OpenStack Resource Controller (declarative management of Keystone services, endpoints, users, and application credentials) | `k-orc/openstack-resource-controller`      |
| **[Labels Injector](https://github.com/cobaltcore-dev/labels-injector)**                     | Automatic Node-to-Pod label synchronization (webhook + controller)                                                                    | `cobaltcore-dev/labels-injector`           |
| **[MariaDB Operator](https://github.com/mariadb-operator/mariadb-operator)**                 | Declarative MariaDB management with Galera clustering and MaxScale                                                                    | `mariadb-operator/mariadb-operator`        |
| **[Memcached Operator](https://github.com/C5C3/memcached-operator)**                         | Kubernetes operator for Memcached instances                                                                                           | `C5C3/memcached-operator`                  |
| **[Memtouch](https://github.com/cobaltcore-dev/memtouch)**                                   | Memory stress test for live migration benchmarks                                                                                      | `cobaltcore-dev/memtouch`                  |
| **[OpenBao](https://github.com/openbao/openbao)**                                            | Centralized secret management and PKI (HashiCorp Vault fork; see [Secret Management](./13-secret-management.md))                      | `openbao/openbao`                          |
| [OpenTelemetry Collector](https://github.com/open-telemetry/opentelemetry-collector)         | Vendor-agnostic telemetry collector for metrics, logs, and traces (CNCF Incubating; see [Observability](./15-observability/))         | `open-telemetry/opentelemetry-collector`   |
| [Prometheus](https://github.com/prometheus/prometheus)                                       | Monitoring system and time-series database (CNCF Graduated; see [Metrics](./15-observability/01-metrics.md))                          | `prometheus/prometheus`                    |
| [provider-kubernetes](https://github.com/crossplane-contrib/provider-kubernetes)             | Crossplane provider for managing Kubernetes resources (Gardener Shoots, ControlPlane CRs)                                             | `crossplane-contrib/provider-kubernetes`   |
| **[Prysm](https://github.com/cobaltcore-dev/prysm)**                                         | Distributed storage observability platform (Ceph, RadosGW, SMART, S3 ops)                                                             | `cobaltcore-dev/prysm`                     |
| **[RabbitMQ Operator](https://github.com/rabbitmq/cluster-operator)**                        | Official Kubernetes operator for RabbitMQ clusters                                                                                    | `rabbitmq/cluster-operator`                |
| [Renovate](https://github.com/renovatebot/renovate)                                          | Automated dependency update tool for source refs and container image tags                                                             | `renovatebot/renovate`                     |
| **[Rook](https://github.com/rook/rook)**                                                     | Storage orchestration for Kubernetes (see [Storage Architecture](./09-storage-architecture.md))                                       | `rook/rook`                                |
| [Syft](https://github.com/anchore/syft)                                                      | SBOM generation tool for container images (CycloneDX, SPDX)                                                                           | `anchore/syft`                             |
| [uv](https://github.com/astral-sh/uv)                                                        | Fast Python package manager (Rust-based), used for container image builds (see [Container Images](./17-container-images/))            | `astral-sh/uv`                             |
| **[Valkey Operator](https://github.com/SAP/valkey-operator)**                                | Kubernetes operator for Valkey (Redis-compatible key-value store) HA with Sentinel mode                                               | `SAP/valkey-operator`                      |
